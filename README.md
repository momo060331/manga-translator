# MangaTL — 英語漫画 日本語翻訳PWA

## 必要なもの

- Anthropic APIキー（https://console.anthropic.com で取得）
- Webサーバー（GitHub Pages / Netlify / Verifyなど無料で使えます）

---

## インストール手順

### ステップ1: ファイルをアップロード

**GitHub Pages（無料・おすすめ）を使う場合:**

1. https://github.com にアクセスしてアカウントを作成
2. 「New repository」で新しいリポジトリを作成
3. このフォルダ内のファイルをすべてアップロード
   - index.html
   - manifest.json
   - sw.js
   - icons/ フォルダごと
4. Settings → Pages → Branch: main → Save
5. `https://あなたのユーザー名.github.io/リポジトリ名/` でアクセス可能に

---

### ステップ2: スマホにインストール

**iPhoneの場合（Safari）:**
1. 上記URLをSafariで開く
2. 画面下の「共有」ボタン（□↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ → ホーム画面にアイコンが追加される

**Androidの場合（Chrome）:**
1. 上記URLをChromeで開く
2. 右上のメニュー（⋮）をタップ
3. 「ホーム画面に追加」をタップ
4. 「追加」をタップ → ホーム画面にアイコンが追加される

**PCの場合（Chrome）:**
1. URLをChromeで開く
2. アドレスバー右端のインストールアイコンをクリック
3. 「インストール」をクリック

---

### ステップ3: APIキーを設定

1. アプリを起動
2. 右上の「⚙」（設定）をタップ
3. Anthropic APIキーを入力（sk-ant-... で始まる）
4. 「保存」をタップ

---

## 使い方

1. 漫画画像のURLを入力して「翻訳」をタップ
   または「タップして画像をアップロード」から画像を選択
2. AIが吹き出しを検出して日本語翻訳を重ねて表示
3. 次のページのURLを入力すれば続けて翻訳できる
4. 翻訳済みのページは「前へ」「次へ」で履歴をたどれる

---

## 注意事項

- 漫画サイトによってはCORS制限でURLからの直接読み込みができない場合があります
  その場合は画像を保存してからアップロードしてください
- APIキーはお使いのスマホのローカルに保存されます（サーバーには送信されません）
- 翻訳にはAnthropicのAPIを使用するため、API利用料金が発生します
